乾坤框架—API

万年历API

# 接口地址

http://000081.xyz/api/wnl.php

# 返回格式

JSON

# 请求方式

GET

# 参数列表

请求参数

字段 必填 说明
type 是 0为阳历日期，1为阳历日期
year 是 请求日期的年份
month 是 请求日期的月份
day 是 请求日期的日数
show 否 填1则返回json键名的中文名

返回参数

字段 返回值 说明
code 200 请求成功
code 404 请求失败
type 阳历 date字段返回日期类型
type 阴历 返回date字段的日期类型
date 日期 返回type字段的日期
text JSON内容 返回请求JSON内容结果

# 请求示例

简洁模式：
    阴历：2021年8月3日
    http://000081.xyz/api/wnl.php?type=1&year=2021&month=8&day=3

    阳历：2021年9月9日
    http://000081.xyz/api/wnl.php?type=0&year=2021&month=9&day=9

详细模式：
    阴历：2021年8月3日
    http://000081.xyz/api/wnl.php?type=1&year=2021&month=8&day=3&show=1

    阳历：2021年9月9日
    http://000081.xyz/api/wnl.php?type=0&year=2021&month=9&day=9&show=1
