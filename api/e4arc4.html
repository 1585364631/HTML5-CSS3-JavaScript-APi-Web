乾坤框架—API

易安卓RC4算法API

# 接口地址

http://000081.xyz/api/e4arc4.php

# 返回格式

JSON

# 请求方式

GET

# 参数列表

请求参数

字段 必填 说明
type 是 0为加密，1为解密
key 是 加解密密钥
text 是 加解密文本

返回参数

字段 返回值 说明
code 200 请求成功
code 404 请求格式错误
type RC4加密 返回执行的操作
type RC4解密 返回执行的操作
key 加解密密钥 返回加解密密钥
text 加解密文本 返回操作前的加解密文本
result 加解密文本 返回操作后的加解密文本

# 请求示例

加密模式：
    密钥：123  文本：123
    http://000081.xyz/api/e4arc4.php?type=0&key=123&text=123

解密模式：
    密钥：123  文本：62c28c
    http://000081.xyz/api/e4arc4.php?type=1&key=123&text=62c28c

# 返回内容

加密模式
    {"code":200,"type":"RC4加密","key":"123","text":"123","result":"62c28c"}

解密模式：
    {"code":200,"type":"RC4解密","key":"123","text":"62c28c","result":"123"}
